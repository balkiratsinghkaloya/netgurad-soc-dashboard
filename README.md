# 🔐 NetGuard — Network Security Operations Center

A real-time Network Security Monitoring Dashboard built as part of a cybersecurity portfolio.

## 🚀 Live Demo
[Click here to view](https://balkiratsinghkaloya.github.io/netgurad-soc-dashboard/)

## 📌 Features
- **Live Packet Capture** — Real-time TCP/UDP/HTTP/DNS/SSH traffic simulation with protocol filtering
- **Port Scanner** — Simulates Nmap-style scanning with risk classification (OPEN / CLOSED / FILTERED)
- **IDS/IPS Alert Engine** — Detects SSH brute-force, ARP spoofing, DNS exfiltration, ICMP floods
- **Traffic Analytics** — 60-second live inbound/outbound chart + geographic origin tracking
- **Network Topology Map** — Visual Internet → Firewall → Gateway → Server layout

## 🛠️ Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js for traffic visualization
- Simulated network layer (Layer 3/4 protocols)

## 🔍 Concepts Demonstrated
- Network protocol analysis (TCP, UDP, ICMP, DNS, SSH, HTTP)
- Intrusion Detection System (IDS) logic
- Port scanning methodology (similar to Nmap)
- Firewall and network topology understanding
- Real-time data visualization for SOC environments

## 📁 How to Run
Just download and open `netguard-soc-dashboard.html` in any browser. No installation needed.
